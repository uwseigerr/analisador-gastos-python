📊 Analisador de Gastos Mensais em Python
Descrição: Sistema desenvolvido para automatizar a leitura de extratos bancários e gerar relatórios visuais de gastos por categoria.

---

Tecnologias:

Python 3.14: Linguagem base.

Pandas: Para manipulação e limpeza de dados.

Matplotlib: Para geração de gráficos de barras e pizza.

---

O que aprendi/apliquei:

Configuração de variáveis de ambiente e Path no Windows.

Tratamento de erros e exceções em Python (try/except).

Modularização de código através de funções.

Transformação de dados brutos em insights visuais.
